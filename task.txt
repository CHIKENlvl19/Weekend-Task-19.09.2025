Процедурное программирование. Два языка С++ по умолчанию + Rust || Go

1. Реализовать Массив. Операции: создание, добавление элемента (по индексу и в конец массива), 
получение элемента по индексу, удаление элемента по
индексу, замена элемента по индексу, длина массива, чтение.

2.Реализовать односвязный список.  
Операции: добавление и удаление элемента (4 способа (до после голова хвост)), 
чтение (разное, несколько способов), 
удаление элемента по значению, поиск элемента по
значению.

3.Реализовать двусвязный список.  Операции: добавление и удаление элемента (4 способа (до после голова хвост)), 
чтение (разное, несколько способов), удаление элемента по значению, поиск элемента по
значению.

4. Реализовать cтек. Операции: добавление и удаление элемента (push и pop),
чтение.

5. Самостоятельно изучить очереди. Открыть книги, которые просила найти на первой лекции.

Везде реализовать вывод, предусмотреть вариант пустых контейнеров.
